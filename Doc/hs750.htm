Receive Program from CNC



Selecting Receive opens a file dialog where you select the name of the file to be received from your CNC Machine Tool.

Once the file to be received is selected, CodeShark  presents a dialog which allows for communication settings to be changed. 
Selecting Receive File performs the file transfers in background, allow you to use CodeShark to perform other tasks.
 


During the receive operation a window is displayed showing the status of the current transfer.



The Receiving Timeout counter displays a count down in seconds at which time the port will be closed and transmission considered complete.
The timeout is set to the value specified in Protocol Options, ASCII EOF Timeout.



If Open On Receive is checked, once the file transfer is complete, CodeShark will open the received file in the editor window.

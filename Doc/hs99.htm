CodeSharkFCs

CodeSharkFCs is an open source editor for your CNC programs and communication software for your CNC controller.
CodeSharkFCs is also a FreeCAD intergration project. The current version provides a mechanism to export design geometry
 (points, lines, arcs, circle center points) into the editor window

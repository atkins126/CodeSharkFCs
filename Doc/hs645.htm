Trace Log



Checking - Enable Dispatcher Tracing tells the communication software (AsyncPro) to generate a Dispatcher Trace log file.
Selecting  - View Trace File opens the Notepad editor with the latest Trace Log file.

More information on the contents and meaning of the data output to the Dispatcher Trace Log file can be found in the 
AsyncPro Reference guide:  http://tpapro.sourceforge.net/docs/APRO_ReferenceGuide.pdf
